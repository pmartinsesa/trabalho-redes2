Relatório

O trabalho foi implementado utilizando a linguagem javascript junto com o framework express para realizara a comunicação entre as entidades

A comunicação inicia com o cliente fazendo uma requisição da temperatura de uma das 3 localizações disponíveis. A cache recebe essa mensagem e checa se tem em seu domínio um dado que esteja com um tempo de vída menor ou igual a 30 segundos, caso não tenha ela faz a requisição do dado para o servidor correspondente e depois retorna o dado pro cliente.
